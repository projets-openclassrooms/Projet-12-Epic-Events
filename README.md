Project Overview
EpicEvent is a Django web application designed to create, manage, and promote events. This project aims to provide a user-friendly platform for event organizers to efficiently plan and execute their events.

## Project Setup
### Prerequisites:

* Python 3.x
* Django
* SGBD : (PostgreSQL, MySQL, SQLITE,MongoDB)

### Installation:

* Clone the repository
* python -m venv venv
* source venv/bin/activate  # On Windows: venv\Scripts\activate
* pip install -r requirements.txt
